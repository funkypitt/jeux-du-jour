Jeux du Jour

French daily word games — Le Mot, L'Hexagone, Connexions. Offline, no ads.

Overview

Jeux du Jour is a collection of three French daily word games inspired by the New York Times puzzle games. Every player gets the same puzzle on the same day, generated deterministically from a date-based seed — no server needed. Native Android (Kotlin / Jetpack Compose).

Features

  • Le Mot (Wordle) — guess the 5-letter French word in 6 tries, with colour-coded feedback
  • L'Hexagone (Spelling Bee) — form as many words as possible from 7 letters; every word must use the centre letter; climb from Débutant to Génie
  • Connexions (Connections) — sort 16 words into four groups of four; four difficulty tiers from yellow to purple
  • Daily puzzles seeded by date — same puzzle for everyone, same day
  • Dictionary powered by Lexique383 (109,000+ French words)
  • 30 hand-curated Connexions puzzles included
  • No ads, no tracking, no accounts, fully offline

Build & install

Standard Android Gradle project.

./gradlew assembleRelease
# or
./gradlew bundleRelease

The Lexique383 SQLite asset is checked in under app/src/main/assets/databases/.
